# MagiWorld
MagiWorld is a fantasy fighting game in 1 VS 1.

Players can choose to be a warrior, a mage or a rover.
Each of these classes has a basic attack and a special attack that is different from the others. 

<ul> 
	<li> Warrior </li> 
		<ul>
			<li> Uses Strength </li>
			<li> Basic attack : Sword Strike -> Causes damages equals to the strength</li>
			<li> Special attack : Rage -> Causes twice the strength as damages, the player receive half the strength of damages</li>
		</ul>
	<li> Mage </li>
		<ul>
			<li> Uses intelligence </li>
			<li> Basic attack : Fireball -> Causes damages equals to intelligence </li>
			<li> Special attack : Heal -> Gives twice the intelligence of HP to the player</li>
		</ul>
  	<li> Rover </li>
  		<ul>
  			<li> Uses agility </li>
			<li> Basic attack :  Bow Shoot -> Causes damages equals to the agility</li>
			<li> Special attack : Focus -> Gives half the level of agility to the player</li>
		</ul>	
</ul>

After choosing the class of your character, you can define its statistics.

* Level (1 - 100)
* Strength (0 - 100)
* Intelligence (0 - 100)
* Agility (0 - 100)

/!\ The total of stat points has to be equal to the chosen level !
